# Custom Fonts

## Albra Display TRIAL Light

To use the Albra Display TRIAL Light font in the app:

### Step 1: Obtain the Font File

**Option A: Purchase/Download from Fontshare**
- Visit: https://www.fontshare.com/fonts/albra
- Download the font family
- Look for the file: `AlbraDisplay-Light.ttf` or `AlbraDisplay-Light.otf`

**Option B: Trial Version**
- If you have the trial version, use the light weight variant

### Step 2: Add Font File to This Directory

1. Copy your font file to: `/Users/imacpro/new1/assets/fonts/`
2. Rename it to: `AlbraDisplay-Light.ttf` (or `.otf`)
3. The app will automatically load it

### Step 3: Restart Expo

```bash
# Stop your current Expo server (Ctrl+C)
# Clear cache and restart
npm start --clear
```

## Alternative Free Fonts (Similar Style)

If you don't have access to Albra Display, here are similar alternatives:

1. **Outfit** (Free on Google Fonts)
   - Very similar geometric sans-serif
   - Download: https://fonts.google.com/specimen/Outfit

2. **Poppins Light** (Free on Google Fonts)
   - Clean, modern, geometric
   - Download: https://fonts.google.com/specimen/Poppins

3. **DM Sans** (Free on Google Fonts)
   - Minimal, elegant
   - Download: https://fonts.google.com/specimen/DM+Sans

To use an alternative, just download the `.ttf` file and place it in this folder with an appropriate name.
